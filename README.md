# weather-app
## Description
A responsive web application that retrieves and displays real-time weather data using a public API. 
The app allows users to search for a location and view current weather conditions in a clean, user-friendly interface.

## Features
- Real-time weather data retrieval via API
- Location-based search
- Dynamic UI updates
- Error handling for invalid inputs
- Responsive design

## How to Run the Project
1. Clone the repository:
git clone https://github.com/yourusername/weather-app.git

2. Open the project folder
3. Open `index.html` in a web browser  
*(or run via a local server if required)*
